# 🌸 Nadee Flora House - Live Order & Delivery Tracker Dashboard

An ultra-modern, production-ready, and mobile-responsive Live Delivery Stepper Dashboard engineered for the floral industry. This core frontend module is part of the next-generation digital transformation ecosystem for Nadee Flora House , designed and developed by ST Imagix.

🚀 Live Demo & Presentation Features
This architecture is optimized for web deployment and later mobile integration via WebViews (React Native / Flutter). It contains an embedded **Admin Demo Control Panel
at the bottom to test and showcase real-time state changes during technical reviews.

---

## ✨ Key Features & Engineering Excellence

- **👑 Dual-Layer Branding:** Premium floral-themed visuals representing *Nadee Flora House* with the signature innovative footprint of *ST Imagix*.
- **💎 Glassmorphic Architecture:** Stunning visual hierarchy utilizing advanced CSS backdrop-blur layers, dynamic responsive borders, and rich gradients.
- **🛡️ Robust State Management:** Powered by an encapsulated, defensive Vanilla JavaScript application state layout that prevents namespace pollution.
- **💾 LocalStorage Persistence:** Automatically caches and maintains the order state upon browser reloads or accidental navigation.
- **🎨 Tailwind CSS v3 Integration:** Lightweight utility implementation paired with a custom CSS layer for premium animations.
- **⚡ Performance Optimized:** Uses 100% Inline SVGs for fluid vector rendering with GPU-accelerated micro-interactions (no external image broken links).
- **🌗 Native Dark/Light Mode:** Seamless context-aware theme toggling that automatically forces background and text transitions.
- **🔊 Mock Web Audio Feedback:** Built-in dynamic sound synthesis utilizing the Web Audio API for interactive audio alerts upon milestone updates.

---

## 🛠️ Tech Arsenal (Tech Stack)

* **Structure:** Semantic HTML5 (strict screen-reader and SEO architecture)
* **Styling:** Tailwind CSS v3 (via CDN) & Vanilla CSS Custom Properties
* **Logic Engine:** Vanilla JavaScript (ES6+ Object-Oriented Model)
* **Icons & Animation:** Embedded GPU-Accelerated Inline SVGs
